Brief

Vous devez réaliser un site pour un concessionnaire automobile. Votre site doit présenter leur dernier modèle, avec ses différentes caractéristiques.
Une partie de la présentation du produit doit se faire sous la forme d'un carroussel.
Le site doit aussi permettre sur une seconde page, de demander une prise de contact via un formulaire par le concessionnaire pour essayer le véhicule.
Ce formulaire doit permettre d'enregistrer le client dans les contacts d'un compte Mailchimp (envoi des données en Javascript).
Il doit aussi permettre d'enregistrer la demande dans un fichier sur votre poste en local via l'appel à du code PHP hébergé sur un serveur en local.

Contexte

Avoir un site web fonctionnel et navigable dans différentes pages.
Le site correspond au wireframe et maquette imaginés.
Le formulaire permet l'enregistrement de données dans Mailchimp.
Il permet également l'enregistrement de la demande dans un fichier en local.

Premier projet de la formation réalisé en duo qui nous a permis de nous initier à JavaScript et PHP.
