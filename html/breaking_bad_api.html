Brief du TD

Connectez-vous à une API à fin d'y récupérer les informations choisies.
Traitez-les dans le but de créer un site d'informations pour les fans.

Contexte

Trouvez le moyen de vous connecter à une API via une requête XMLHttpRequest.
Parcourez et récupérez les données pour les réexposer sur votre site à l'aide du Mockup fourni.

TD d'abord réalisé en JS Vanilla puis en jQuery

Technologies
